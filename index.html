CHAPTER 01 · Shader 概述

PBR_DYE SHADER

pbr_dye 染色着色器

Shader 名称

pbr_dye

基于 PBR 的染色着色器，在渲染控制中开启染色后生效

核心能力
  • ● 支持最多 4 个染色分区
  • ● 3 种染色模式：单色 / 双色渐变 / 多彩渐变
  • ● 2 种渐变方式：方向 / 贴图
  • ● 独立调节色相/饱和度/亮度/金属度/粗糙度
Fig.01 渲染控制中开启染色
pbr_dye 是基于 PBR（基于物理的渲染）的染色着色器。使用前需要在渲染控制中开启染色功能。

核心能力：4 分区、3 种染色模式、2 种渐变方式，以及独立的颜色和材质参数调节。
CHAPTER 02 · 染色分区
04 / 14
DYE PARTITIONING · 遮罩分区

通过遮罩图控制 4 个分区

同一个材质支持 4 个分区，分别是遮罩图中的黑色、红色、绿色和蓝色区域。
注意：Alpha 通道另有用处（贴图渐变时使用），不参与分区。

分区 0

黑色区域

RGB = (0,0,0)

分区 1

红色区域

RGB = (255,0,0)

分区 2

绿色区域

RGB = (0,255,0)

分区 3

蓝色区域

RGB = (0,0,255)

UV SCALE

如果使用 1U 去分区，缩放偏移要设置为 1, 1, 0, 0（即 x=1, y=1, u=0, v=0）

染色分区是染色系统的基础概念。

遮罩图使用 RGB 通道来区分染色区域：
• 黑色 (0,0,0) → 分区 0 • 红色 (255,0,0) → 分区 1
• 绿色 (0,255,0) → 分区 2 • 蓝色 (0,0,255) → 分区 3

⚠️ Alpha 通道不参与分区，它用于贴图渐变功能。
⚠️ 如果使用 1U 去分区，缩放偏移必须设置成 1, 1, 0, 0。

如何制作遮罩图

在 Photoshop 或 SP 中创建纯色图层，用不同颜色填充不同区域。每个颜色对应一个染色分区。

注意事项
  • ▸ 颜色值必须精确，不要有渐变混合
  • ▸ Alpha 通道保持白色（或按贴图渐变需求使用）
  • ▸ 贴图格式为 RGBA，不要用压缩格式
参考 POPO 文档 · 美术设置 → 染色分区
这张图展示了遮罩图的实际效果。RGB 通道分别用黑/红/绿/蓝四种纯色来区分不同的染色区域。

制作时注意：颜色值必须精确为纯色，不能有渐变或混合。Alpha 通道保持白色。
CHAPTER 03 · 颜色控制
06 / 14
DYE COLOR MODES · 3 种染色状态

分区颜色 3 种模式

通过参数控制：0 单色 · 1 双色渐变 · 2 多彩渐变

0
MODE 0 · 单色染色

去色 → 单色

把基色去色后，使用染色 1 (u_dye_color_major) 颜色进行染色

1
MODE 1 · 双色渐变

去色 → 双色插值

把基色去色后，在染色 1 和染色 2 之间直接插值渐变

2
MODE 2 · 多彩渐变

去色 → 色环插值

把基色去色后，在两色之间沿色环最近距离插值渐变

三种染色模式，通过 u_a_shift_mode 参数控制：

Mode 0 · 单色：基色去色后，直接用 u_dye_color_major 覆盖。
Mode 1 · 双色渐变：基色去色后，在两色之间线性插值。
Mode 2 · 多彩渐变（重点推荐）：插值沿色环最短距离进行，从红到绿会经过黄色而非灰色。

⚠️ 所有模式第一步都是"去色"——基色转灰度再上色。

CHAPTER 04 · 渐变方式
08 / 14
GRADIENT METHODS · 两种渐变

渐变方式

通过参数 u_a_use_gradient_tex 控制：0 方向渐变 · 1 贴图渐变

0

方向渐变

根据物体自身空间位置进行渐变，可设置方向和范围。

Fig.04 方向渐变 · SP灰度矫正
u_shift_direction — 渐变方向
u_shift_range — 渐变宽度
1

贴图渐变

根据遮罩图的 Alpha 通道值在两色间渐变。完全由美术控制。

Fig.05 贴图渐变 · RGB通道mask
▸ Alpha 值 0 → 染色 1 颜色
▸ Alpha 值 1 → 染色 2 颜色
▸ Alpha 值 0.5 → 中间混合
两种渐变方式：

Mode 0 · 方向渐变：根据物体空间坐标渐变，需设置 u_shift_direction 和 u_shift_range。
Mode 1 · 贴图渐变：使用遮罩图 Alpha 通道驱动。Alpha=0→染色1，Alpha=1→染色2。

⚠️ 贴图渐变时，Alpha 通道需根据需求绘制灰度值。
CHAPTER 05 · 程序参数
09 / 14
MODEL STRUCTURE · 模型结构

模型 结构

Fig.06 模型分为 3 个部分，每部分最多 4 个染色区域
结构说明

每个模型由美术分为若干个部分（如上衣/裤子/鞋子），每个部分又可拥有至多 4 个染色区域。

计算方式
3 parts × 4 zones = 12 zones

一个角色最多支持 12 个独立染色区域

模型结构：每个模型由美术分为若干部分，每部分最多 4 个染色区域。

例：3 个部分 × 4 区域 = 12 个独立可控染色区域。可以对角色每个部位的不同区域分别设置颜色、渐变和 HSL。

PROGRAM PARAMETERS

整体参数

参数名类型说明
UES_DYEINGmacro材质宏，需开启才能启用染色
DyeingMaskTexTexture遮罩贴图，RGB 对应 4 个分区
u_mask_scale_offsethalf4遮罩贴图缩放偏移。UV 时设为 1,1,0,0

整体参数是所有染色区域共用的全局设置：

• UES_DYEING：材质宏，必须开启。最容易遗漏的一步。
• DyeingMaskTex：遮罩贴图（黑/红/绿/蓝分区）。
• u_mask_scale_offset：UV 缩放偏移，设为 1,1,0,0。
• u_shift_direction / u_shift_range：方向渐变专用。

CHAPTER 06 · 贴花制作
12 / 14
DECAL PRODUCTION

贴花 制作方法

Fig.07 贴花 · 4通道染色参数
原理

染色带自定义图案，贴图有 Alpha 通道，叠加到原本的贴图上。

制作要点
  • ▸ 图案区域使用 Alpha 通道控制透明度
  • ▸ 非图案区域 Alpha = 0
  • ▸ 叠加到染色区域上形成装饰效果
应用场景

装饰条纹、logo 图案、纹理细节等需要在染色基础上叠加的自定义图案。

贴花是在染色基础上叠加自定义图案的方法。

原理：制作 Alpha 通道贴图，图案部分有颜色和非零 Alpha，非图案区域 Alpha=0，叠加到染色结果上。

常见应用：装饰条纹、logo、纹理细节。
